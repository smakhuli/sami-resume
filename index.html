About This Website

I designed and built this responsive website from scratch using HTML5, CSS3, Javascript and JQuery plugins. I had originally planned to build this website using WordPress, but decided to develop it from scratch to showcase my knowledge of HTML, CSS and Javascript/JQuery. It uses media queries to make the website responsive for the following category of device/browser sizes:

  • 0 to 480 pixels
  • 481 to 767 pixels
  • 768 to 1023 pixels
  • 1124 to 1200 pixels

All code for this website is avaialable for review on GitHub in a public repository. You can access the repository by clicking here.

The purpose of this website is to serve as a replacement for the traditional resume. You can access a more traditional resume by clicking here. This link will take you to a resume that is generated by a Ruby on Rails resume builder application I also developed from scratch. You can access the GitHub public repoistory for the Ruby on Rails web application by clicking here.

I tested this website with the Chrome, Firefox and Safari browsers. I used the https://validator.w3.org webiste to validate my HTML and it passed with zero errors or warnings. I used animate.css to add some animation to the header section of the website. I used the JQuery waypoint plugin to help suppport sticky navigation, smooth scrolling and mobile navigation.
